# CloudRader

Open-source tools for everyday organizational needs

CloudRader is a small collection of open-source applications designed to help organizations manage their internal processes. All tools are self-hostable, simple to deploy, and easy to adapt.

---

## What We Provide

### Reservium — Reservation System

Our first project.
A practical tool for booking rooms, spaces, and shared resources.
Supports SSO, multiple reservation types, and a clear user interface.

### Inventory System (planned)

A lightweight system for tracking equipment and other assets.

### Access Management (planned)

A unified way to manage user access to spaces within an organization.

---

## Our Approach
- Open-source and transparent
- Self-hosting friendly
- Designed to work independently or together
- Planning a cloud-native platform where all tools integrate smoothly
